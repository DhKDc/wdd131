Project Site Plan

Project Name

ExpoAnime Rancagua

Site Purpose

The project is a website that provides information about the anime expos that take place in my city. The website will include details about the upcoming expos, location, schedule, and ticket prices. It will also feature photo galleries from past events and a blog section with news and updates about the anime community.

Scenarios

  • Scenario 1: A user wants to know the date and location of the next anime expo in Rancagua.
  • Scenario 2: A user is interested in buying tickets for the expo and wants to know the prices and where to purchase them.
  • Scenario 3: A user wants to see photos from past expos to get an idea of what to expect.
  • Scenario 4: A user is looking for news and updates about the anime community in Rancagua.

Color Schema

  • Primary Color: #ff0000 (Red)
  • Secondary Color: #000000 (Black)
  • Accent Color: #ffcc00 (Yellow)

Typography

  • Heading Font: Funnel Display
  • Body Font: Arial, sans-serif
