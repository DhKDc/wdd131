Project Site Plan

Project Name

ExpoAnime Rancagua

Site Purpose

The project is a website that provides information about the anime expos that take place in my city. The website will include details about the upcoming expos, location and schedules. It will also feature photo galleries from past events and a blog section with news and updates about the anime community.

Scenarios

  • Scenario 1: A user wants to know the date and location of the next anime expo in Rancagua.
  • Scenario 2: A user wants to see photos from past expos to get an idea of what to expect.
  • Scenario 4: A user is looking for news and updates about the anime community in Rancagua.
